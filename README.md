# 🧬 LLM Semantic Mind Map

**Mapping the Hidden Architecture of AI Meaning**

The **LLM Semantic Mind Map** is a high-performance, interactive visualization engine that compares semantic associations across the world's leading language models—**GPT-4**, **Claude 3**, and **Gemini 1.5**. 

By transforming high-dimensional embedding data into a gorgeous **Radial Graph**, this platform reveals how different AI architectures perceive 20 unique philosophical and technical domains, from *Power* and *Chaos* to *Beauty* and *Technology*.

---

## ✨ Core Features

### 🔍 Intuitive Navigation
- **Semantic Zoom**: Scroll to zoom relative to your cursor. Labels stay readable via an adaptive font-scaling logic.
- **Fluid Panning**: Navigate dense clusters by dragging the graph and moving through the semantic rings.
- **Instant View Controls**: Reset your perspective or fine-tune zoom with dedicated UI buttons.

### 🎭 Multi-Model Comparison
- **Real-Time Toggling**: Instantly switch between GPT-4, Claude 3, and Gemini 1.5. 
- **Consensus Highlighting**: Highly aligned associations identified by multiple models are visually merged and highlighted in **Purple**, signifying "cross-model consensus."
- **Distance-to-Meaning Mapping**: Distance from the center directly maps to cosine similarity. The closer a word is to the center, the stronger the AI's association with the core topic.

### 📊 Deep Analytics
- **Node Inspector**: Click any node to reveal embedding confidence stats and model-specific affinity.
- **Consensus Rankings**: View the top shared associations across all active models in a live-updating dashboard.
- **Rich "Cyber-Industrial" Aesthetic**: A premium UI powered by glassmorphism, neon accents, and custom-engineered Canvas2D animations.

## 🚀 Getting Started

### 1. Installation
Clone the repository and install dependencies using `npm`.

```bash
git clone https://github.com/your-username/llm-mind-map.git
cd llm-mind-map
npm install
```

### 2. Development
Launch the high-performance Vite server.

```bash
npm run dev
```

### 3. Build & Production
Generate a production-ready bundle.

```bash
npm run build
```

## 🛠️ Technical Stack
- **Engine**: Custom Canvas2D rendering (vanilla JS for maximum performance).
- **Frontend**: React and Vite for ultra-fast HMR and state management.
- **State Pattern**: `useRef` mutable-state patterns to handle 60FPS animation loops without React re-render overhead.
- **Data**: Pre-processed JSON datasets derived from raw LLM neighbor outputs.
